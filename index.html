Bus Departure Display

Live board showing upcoming departures from Kilbirnie stops A, B & C — powered by Metlink.

→

This project was created for the EBIS Science Fair. It connects to the Metlink public API to fetch real-time departure data for bus stops near the school, and displays it on a monitor via iPhone screen casting.
